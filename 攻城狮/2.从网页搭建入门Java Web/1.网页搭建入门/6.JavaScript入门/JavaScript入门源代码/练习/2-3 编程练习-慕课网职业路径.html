2-3 编程练习

小伙伴们，尝试用switch语句来编写一段代码吧，实现如下要求：

（1）   页面弹出输入框，提示“请输入您要查询的业务编号”

（2）   输入“1”, 页面显示“账户余额为。。。。。。”

（3）   输入“2”, 页面显示“转账余额为。。。。。。”

（4）   输入“3”, 页面显示“信用额度为。。。。。。”

输入其它数字，页面显示“退出银行系统。。。。。。”


任务

（1）页面弹出输入框，让用户输入要查询的业务编号

（2）用switch语句写这段判断的代码

  当用户输入数字1时，页面显示：账户余额为。。。。。。；

  当用户输入2时，页面显示：转账金额为。。。。。；

  当用户输入3时，页面显示：信用额度为。。。。。；

  当用户输入其他数字，页面显示：退出银行系统。。。。


任务提示

注意：

1、关于弹出框的内容可以查看教辅文档《JS弹出框教辅》

2、用户在页面中输入的数字类型均为字符串
